# Proyecto_Statistical_Learning_1
Repositorio para el Proyecto Final Curso Statistical Learning 1

Este repositorio contiene:
Parte 1 del Proyecto Final de Statistical Learning 1:
  a. Feature Engineering sobre DataSet Titanic.
  b. Entrenamiento de Modelos: Árbol de Decisión, SVM, Naive-Bayes y Regresión Logística con Sigmoid.
  c. Ensemble Learning: Predicción por Votación Mayoritaria
  d. Fase de Experimentación: Exportación de Modelos, Bitácora de Experimentos (métricas de evaluación) sobre DataSet de Entrenamiento y Validación.
  e. Predicción Final sobre DataSet de Prueba (con Accuracy requerido en el proyecto).
Parte 2 del Proyecto Final de Statistical Learning 1:
  a. Importación de Mejores Modelos de la Parte 1.
  b. Ensemble Learning utilizando cuatro modelos y Predicción Final por Votación Mayoritaria.
  c. Función de Predicción con despliegue de Interpretabilidad de Modelos de Árbol de Decisión y distribución probabilística de Naive-Bayes y Regresión Logística.
